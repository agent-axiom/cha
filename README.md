# Пуэр · Живая гора

Интерактивный русскоязычный лонгрид о двух путях пуэра — шэн и шу. Сайт отделяет легенды от письменных свидетельств и современных научных данных, показывает географию Юньнани, производство, микробную ферментацию, медицинские исследования и ограничения доказательности.

После публикации: [agent-axiom.github.io/cha](https://agent-axiom.github.io/cha/)

## Локальный запуск

Требуется Node.js 22 или новее.

```bash
npm install
npm run dev
```

Vite откроет проект по адресу `http://localhost:5173/cha/`.

## Проверка

```bash
npm test
npm run build
npm run test:e2e
```

- Vitest проверяет модель источников, компоненты и axe-аудит.
- Playwright проверяет переключение Шэн / Шу, якорную навигацию, клавиатурный вход, reduced motion и узкий экран.
- Production-сборка создаётся в `dist/` с базовым путём `/cha/`.

Для первого локального запуска E2E может потребоваться браузер:

```bash
npx playwright install chromium
```

## Архитектура

- `src/content/` — типизированная история, производство, медицина, мифология, регионы и библиография.
- `src/components/` — независимые семантические разделы и интерактивные сцены.
- `src/styles/` — визуальная система «Живая гора», адаптивность и reduced motion.
- `public/images/` — четыре оригинальных WebP-визуала.
- `e2e/` — сквозные браузерные сценарии.
- `.github/workflows/deploy.yml` — сборка и публикация GitHub Pages.

Содержательная графика карты и лаборатории реализована в SVG. Canvas используется только для декоративного тумана и отключается при `prefers-reduced-motion: reduce`.

## Источники и редакционные границы

Библиография сочетает китайские исторические тексты и официальные документы, работы азиатских исследователей, западную историографию, PubMed, EFSA, WHO и UNESCO. Каждое существенное утверждение связано с записью в `src/content/sources.ts`.

Цитаты о «ста болезнях» и другие лечебные свойства старых трактатов показаны как история медицинской культуры. Они не являются рекомендацией, диагнозом или доказательством лечебного эффекта. Лабораторные результаты, животные модели и небольшие испытания экстрактов не приравниваются к пользе обычной чашки чая.

## Оригинальные изображения

Четыре растровых изображения созданы специально для проекта с помощью OpenAI Image Generation и затем локально оптимизированы в WebP. В них нет встроенного текста, логотипов или товарных знаков.

| Файл | Замысел |
| --- | --- |
| `puer-hero.webp` | древний чайный лес Юньнани в утреннем тумане |
| `sheng-shou-paths.webp` | один свежий лист между янтарным шэн и рубиновым шу |
| `puer-material-culture.webp` | прессованный чай, волокнистая обёртка, чайный нож и гайвань |
| `fermentation-microcosm.webp` | художественно-научный микромир влажности, тепла и микроорганизмов |

Общее направление промптов: документальная или научно-художественная достоверность, естественный свет, лесной зелёный / земля / чайный янтарь, широкий кадр 16:9, правдоподобные материалы, без текста, логотипов, водяных знаков и псевдомедицинской символики.

## Публикация

Push в `main` запускает GitHub Actions. Workflow устанавливает зависимости через `npm ci`, выполняет unit/axe-тесты, собирает Vite и разворачивает `dist/` в GitHub Pages с правами только на чтение репозитория и публикацию Pages.

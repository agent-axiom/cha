# Издательский контур книги «Пуэр. Живая гора»

Работа над книгой ведётся в фиксированной последовательности:

1. Валидировать исходные данные командой `npm run book:validate:data`.
2. Подготовить черновик, отмечая проверяемые утверждения маркерами источников (`claim markers`).
3. Закрыть редакционные, фактологические и экспертные рецензии.
4. Получить и зафиксировать права на все тексты, изображения, карты и другие материалы.
5. Собрать редакционную пробу (`proof`).
6. Отрендерить пробу в постраничный формат.
7. Визуально проверить страницы: композицию, переносы, подписи, изображения, поля и вылеты.
8. Получить у выбранной типографии актуальные технические требования.
9. Экспортировать финальные файлы печати в требуемом типографией стандарте PDF/X.
10. Выполнить предпечатную проверку (`preflight`) экспортированных PDF/X.

Редакционная проба и финальный печатный PDF/X — разные артефакты. Проба предназначена для согласования содержания и визуальной проверки и не считается готовым файлом для типографии. Финальный PDF/X создаётся только после закрытия рецензий и прав, получения требований типографии и исправления замечаний к пробе; именно экспортированный PDF/X проходит итоговый preflight перед передачей в печать.
